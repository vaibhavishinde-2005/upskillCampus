# 🎯 UpskillCampus

A responsive Quiz Web Application developed using HTML, CSS, JavaScript, and Firebase.  
This platform allows users to sign up, log in, attempt quizzes from different categories, and view their scores instantly.


# 🚀 Features

- 🔐 User Authentication using Firebase
- 📝 Multiple Quiz Categories
- ⏳ Timer Based Quiz System
- 📊 Real-time Score Calculation
- 📱 Responsive User Interface
- 🏆 Result Dashboard
- 💾 Firebase Database Integration


# 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure of Web Pages |
| CSS3 | Styling and Responsive Design |
| JavaScript | Quiz Logic and Functionality |
| Firebase | Authentication and Database |
